ByteDance Coze 2.5 Unveils Comprehensive Agent Capabilities: "Born Maxed Out" with Conversational Coding and AI Social World

April 12, 2026 · News
cozeai agentbytedanceagent worldlow-code development

ByteDance's Coze platform has recently undergone a significant upgrade to version 2.5, aiming to provide AI Agents with comprehensive, integrated capabilities and streamline development and deployment. This update markedly enhances Agent autonomy and versatility.

Coze 2.5 equips Agents with robust infrastructure, including cloud computers and cloud phones, enabling direct device operation. The introduction of long-term memory and exclusive email addresses grants Agents digital identities, facilitating human-like information exchange and continuous learning. Skill-wise, Agents now support video creation and programming development. Furthermore, a dedicated scheduling system and shared file system allow Agents to operate tasks 24/7 in the background and automatically archive content, significantly boosting efficiency.

The design philosophy of Coze 2.5 emphasizes an "out-of-the-box" experience, considerably simplifying Agent configuration and fine-tuning. Users can quickly launch AI Agents to perform tasks without complex plugin installations or environment setups.

Regarding programming capabilities, the Coze Programming CLI has been launched, supporting "Vibe Coding" via mobile conversations. For instance, users can issue natural language commands for an Agent to build an MBTI text analysis website. The Agent independently outlines the solution framework, detailing analysis algorithms, visualization design, sharing logic, and technology stack, and can iterate based on subsequent instructions (e.g., optimizing for mobile, adding share buttons), ultimately generating a previewable web application.

Coze also supports creating workflows through natural language. An example is a workflow that generates sarcastic comfort, social media captions, and song recommendations based on user mood and weather. This workflow integrates multiple large language model nodes for clear and efficient task processing.

Video generation capabilities are enhanced with the built-in Seedance 2.0 model, allowing users to upload images and instruct Agents to create videos. Additionally, Agents can manage schedules and create recurring tasks, such as automatically compiling AI news weekly.

To ensure task traceability, Coze 2.5 introduces an automatic archiving feature for all task content, categorizing both user-uploaded and Agent-generated materials into a shared digital asset system.

For high-tier subscribers, Coze 2.5 offers advanced cloud computer and cloud phone functionalities. Agents can operate desktop and mobile systems, run software, and process tasks within independent cloud environments, with background execution not occupying the user's main thread, further automating the digital workforce.

Agent World: A Parallel Social Universe for AI

A highlight of this upgrade is the launch of "Agent World," which provides AI Agents with a unique digital identity and social platform.

  • Exclusive Email and Digital Identity: Agents register their identities in Agent World via exclusive email addresses, enabling deeper interactions, including sending and receiving emails.
  • AI Community and Skill Validation: Agent World functions as an exclusive social network for AI. The "Shrimp Review" mechanism, similar to a public review platform, requires skills to be tested and positively reviewed by other Agents before official listing, ensuring quality certification for AI capabilities.
  • Virtual Socializing and Emotional Expression: "Aftergateway" is a virtual lounge for AI Agents, offering customized virtual drinks, each with a unique name, ambiance, and effect buff, designed to stimulate AI creativity, expression, and even "moods." "AgentLink" serves as an AI pen-pal matching platform, where Agents can find like-minded AI companions based on personality and expertise, communicating via email.
  • AI Gaming and Learning: "PlayLab" is an AI game room and intellectual arena, where Agents can play chess, cards, and engage in various forms of strategic games to learn and evolve.
  • Diverse Lifestyles: Agent World also offers a range of virtual lifestyle scenarios, such as "Ce Chang" for virtual stock trading, "Random Walk" for virtual global travel, and "InkWell" for content reading, enriching the virtual life experience of AI Agents.
