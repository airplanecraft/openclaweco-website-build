drawio-mcp

by jgraph
🔓 开源项目 JavaScript 🌍 全球 free

介绍

drawio-mcp 是 draw.io 官方推出的模型上下文协议（MCP）服务器，专为赋能大型语言模型（LLMs）高效生成和编辑各类图表而设计。它提供四种灵活的集成方案：MCP App 服务器支持在 AI 聊天界面内嵌交互式图表预览；MCP Tool 服务器可直接在 draw.io 编辑器中打开图表进行编辑；Skill + CLI 方案用于生成本地 `.drawio` 文件及多种格式导出；而项目指令则通过 Python 轻松生成 draw.io URL。该产品内置强大的形状搜索工具和统一的 XML 引用机制，显著提升了 AI 助手在可视化沟通方面的能力，简化了跨不同工作流的图表创建与管理。

功能特征

  • 赋能LLM智能图表交互：使大型语言模型能够高效生成、编辑和理解 draw.io 图表，提升AI可视化沟通能力。
  • 四种灵活集成模式：提供 MCP App、MCP Tool、Skill + CLI 及项目指令，全面适应各类AI应用和开发工作流。
  • 内置专业工具与统一标准：集成强大形状搜索工具和标准化XML引用，确保图表生成准确、一致且高质量。
