AWS开售OpenAI模型，微软独家协议终结，AI市场格局巨变

2026年4月29日 · News
openaiawsmicrosoft azuregpt modelsai agents

亚马逊网络服务（AWS）于周二宣布，将开始向其云客户销售OpenAI模型。此举发生在前一天微软同意终止其与OpenAI为期三年的独家转售协议之后，该协议在生成式AI时代初期赋予了Azure对OpenAI技术的独家访问权。AWS首席执行官Matt Garman在接受彭博电视台采访时表示：“我们的客户长期以来一直有这个需求。”

OpenAI的部分最新模型已于周二开始在AWS上提供预览，而更强大的GPT模型也将在未来几周内陆续上线。这一声明标志着AI行业内这一最重要伙伴关系的重组完成，该重组始于今年2月，当时亚马逊承诺向OpenAI价值1100亿美元的融资轮投入高达500亿美元，这笔交易使ChatGPT制造商的估值达到8520亿美元，并成为亚马逊有史以来对单一公司最大的投资。此外，OpenAI也承诺在未来八年内，将在AWS的计算能力和Trainium芯片上投入1000亿美元，消耗高达两千兆瓦的容量。值得注意的是，两家公司还在Bedrock上共同为智能体AI构建了一个有状态运行时环境（Stateful Runtime Environment）。

这项合作重组的关键问题并非OpenAI的模型是否足够优秀，以至于可以在竞争对手的云平台上销售。真正的问题是，OpenAI能否在任何地方销售足够多的模型，以证明其所承诺的巨额支出是合理的。特别是在《华尔街日报》报道OpenAI未达到收入和用户目标、预计250亿美元现金消耗对比300亿美元收入，以及对AWS、Azure和Oracle数百亿美元的基础设施承诺，而这些都基于OpenAI尚未证明的增长前景时，这一挑战显得尤为突出。

此次重组还涉及微软与OpenAI合作伙伴协议的修订。周一修订的协议取消了此前定义AI繁荣第一阶段的独家安排。微软对OpenAI模型和产品知识产权的授权，已从独家变为非独家，但延长至2032年。微软将保留OpenAI 20%的收入分成直至2030年，但现已设定上限。根据新协议，OpenAI仍必须首先在Azure上发布其模型，除非微软无法或选择不支持所需功能。这一条款保留了Azure的早期访问优势，同时解除了此前阻止AWS和Google Cloud客户通过其现有云提供商访问OpenAI技术的限制。此外，重组还移除了“AGI条款”，这项法律上不寻常的条款规定，如果OpenAI董事会认定公司已实现通用人工智能（AGI），微软的商业权利将终止。该条款此前已成为治理负担而非实际保障，其移除表明两家公司都已接受了新的合作关系模式。
