Cloudflare发布EmDash：AI Agent优先的建站平台，欲取代WordPress？

2026年4月10日 · News
ai agentsemdashmodel context protocolwordpresstypescript

作为连接数百万网站的领先云服务提供商，Cloudflare近日推出了EmDash，一个全新的开源系统。EmDash旨在解决“WordPress无法解决的核心问题”，其核心理念是允许AI Agent直接控制和管理网站。Cloudflare将EmDash定位为WordPress的“精神继任者”，目前该系统正处于早期测试阶段，已在WordPress社区引发广泛关注和讨论。

Cloudflare宣称EmDash是“从零开始”重建的开源WordPress项目。其关键创新在于内置了模型上下文协议（MCP）服务器，使得大型语言模型（LLM）能够轻松连接并与平台文档进行交互。EmDash基于Cloudflare为LLM优化的Web构建框架Astro运行，并采用TypeScript编程语言，这种语言更便于AI Agent理解。此外，EmDash还支持x402协议，允许网站发布者向访问其内容的AI爬虫收取费用。

EmDash的发布引发了褒贬不一的反应。WordPress.com所有者Automattic的开发者倡导者Brian Coords赞扬了EmDash快速建站的能力，称“从零到基本设计非常快，真的非常快。”但他也指出该平台“感觉有点随意编码”。WordPress创始人Matt Mullenweg持有类似观点，他在博客中描述EmDash的界面处于“有点像WordPress又有点不像的恐怖谷效应”，并认为EmDash的推出主要是为了“销售更多的Cloudflare服务”。Mullenweg还明确反驳了“精神继任者”的说法，强调“在不理解我们精神的情况下，请不要自称是我们的精神继任者。”尽管Mullenweg承认EmDash的AI驱动技能功能是“一个不错的亮点”，但他并未深入探讨WordPress自身存在的深层次问题。

除了Mullenweg的质疑，EmDash的出现也在WordPress社区内部引发了更广泛的讨论。虽然部分批评与Mullenweg的观点相似，但也有人借EmDash的发布，强调WordPress项目自身需要改进的领域，尤其是在其架构、安全性以及对AI技术的适应性方面。知名WordPress插件Yoast的创始人Joost de Valk称EmDash为“多年来内容管理领域最有趣的事情”。他特别赞赏EmDash对AI Agent的原生支持及其结构化内容设计，使得“机器能够轻松解析和操作”。de Valk进一步指出，EmDash“暴露了WordPress现有编辑器Gutenberg的一个老旧弱点”，即Gutenberg以HTML格式存储数据，这种格式对机器处理不够友好，这一观点也与开发者Hendrik Luehrsen的担忧不谋而合。

EmDash

EmDash 是一个基于 Astro 和 Cloudflare 构建的现代化全栈 TypeScript CMS。它重新构想了 WordPress 的可扩展性和用户体验，将其置于无服务器、类型安全的架构之上。核心亮点是其创新的沙盒插件生态系统，插件在独立的 Cloudflare Worker 沙盒中运行，具备明确的能力声明，从根本上解决了传统 CMS 的安全隐患。EmDash 采用 Portable Text 进行结构化 JSON 内容存储，实现内容与表现层的解耦，支持多渠道分发。此外，它专为 AI Agent 设计，提供开发技能文件和程序化 CLI 工具，并内置 MCP 服务器，支持 AI 工具直接交互。EmDash 兼容多种数据库和 S3 存储，为开发者提供一个高性能、安全且面向未来的内容管理平台。

maxun

Maxun 是一个功能强大的开源无代码平台，旨在将任何网站转化为结构化、可靠的数据或实时API。它集成了实时网络爬取、智能抓取、自动化搜索和先进的AI数据提取等核心功能，帮助用户轻松实现网站数据的自动化收集。通过直观的“录制模式”，用户可以记录操作行为以创建可复用的提取机器人；利用LLM驱动的“AI模式”，只需自然语言描述即可智能提取所需数据。Maxun 还支持将网页内容转换为简洁的Markdown或HTML格式，处理复杂的分页与滚动，并能按计划运行自动化任务。无论是用于市场研究、内容聚合还是潜在客户开发，Maxun 都提供了一套灵活且可自部署的解决方案，无需编写代码即可高效管理和利用网络数据。

firecrawl

firecrawl 是专为 AI 设计的强大网页数据 API，旨在赋能 AI 智能体获取高质量、结构化且实时的网络数据。它提供行业领先的搜索、抓取和交互能力，支持大规模网页数据处理，覆盖高达96%的网页内容，包括复杂JavaScript渲染页面。firecrawl 能够输出对大型语言模型（LLM）友好的干净Markdown、结构化JSON及截图等格式，显著提升AI应用的开发效率和数据质量。开发者无需关注代理轮换、速率限制、JS阻碍等复杂问题，即可轻松集成并赋能AI智能体和动态应用，实现实时、精准的网络信息获取和交互。
