荷兰政府推出开源代码平台，强化数字主权

2026年4月29日 · News
forgejogiteagitopensourcedigitalsovereignty

荷兰政府近日宣布，已正式推出其专属的开源代码托管平台 code.overheid.nl。此举标志着该国在数字基础设施建设和政府IT现代化方面迈出了重要一步。

据介绍，该平台的技术基石是 Forgejo。Forgejo 是一个由 Gitea 分支而来的开源项目，其设计理念与功能与广为人知的 GitHub 高度相似。它不仅提供了强大的 Git 软件开发与版本控制能力，还集成了项目管理所需的核心功能，包括详细的 Bug 跟踪系统、协作式的 Wiki 文档以及严格的代码审查流程，旨在为政府开发者提供一个高效、安全的协作环境。

code.overheid.nl 平台完全托管于荷兰政府自身的数字基础设施之上，确保了数据的安全性和主权。更重要的是，该平台将向荷兰境内的所有政府机构免费开放，方便各部门共享代码资源、促进跨部门协作。荷兰政府明确指出，推出这一平台是其加强国家数字主权战略的重要组成部分，旨在减少对外部商业服务的依赖，提升自主可控能力。
