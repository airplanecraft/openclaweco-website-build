OpenAI拟出售Codex：探索战略调整？

2026年4月21日 · News
openaicodexai modeldivestituregenerative ai

据新浪财经最新消息，全球领先的人工智能研究与部署公司OpenAI目前正与专业的顾问团队紧密合作，积极探寻其人工智能编程模型Codex的战略方案，其中包括对其进行潜在出售的可能性。

Codex是OpenAI开发的一款先进语言模型，其核心功能在于将自然语言指令高效地转化为可执行的代码，是GitHub Copilot等广受欢迎的代码生成工具的技术基石。此次探索出售Codex的举动，可能预示着OpenAI正在对其内部资产组合或未来发展战略进行重大调整。

CLIProxyAPI

CLIProxyAPI 是一款专业的代理服务器，旨在为开发者提供统一、高效的命令行界面（CLI）大型语言模型访问方案。它无缝封装 Gemini CLI、Antigravity、ChatGPT Codex、Claude Code、Qwen Code 和 iFlow 等服务，转换为兼容 OpenAI/Gemini/Claude/Codex 的标准化 API 接口。开发者因此能便捷地通过现有订阅或 CLI 访问主流 LLM。核心功能涵盖多账户负载均衡、支持 OAuth 认证的 Codex 与 Claude Code、多模态输入、函数调用及流式响应。此外，产品深度集成 Amp CLI 及 IDE 扩展，提供智能路由与模型回退，并附带可复用 Go SDK，显著提升 AI 编程与应用开发效率。

openai-agents-python

OpenAI Agents SDK 是一个轻量且强大的 Python 框架，用于构建多 Agent 工作流。它支持 OpenAI API 及其他100多种大型语言模型，通过配置Agent、工具、安全防护和会话管理等核心概念，实现复杂的智能体协作和任务自动化。其特点包括Agent即工具、沙盒环境运行、内置人类干预、会话历史管理和运行追踪，支持构建语音Agent。
