Anthropic悄然上调Claude Code代币成本预估：开发者支出翻倍

2026年4月29日 · News
anthropicclaudetokensai agentcost

AI公司Anthropic近期悄然将其Claude Code产品的代币成本预估翻了一倍多，这直接影响了开发者和企业用户的支出预期。

Anthropic最新估计，平均每个开发者每天在Claude Code代币上的花费约为13美元。而在本月初，这家AI初创公司公布的每日预估成本是6美元，这意味着成本预期在短短几周内翻了不止一倍。

根据Claude Code官网的描述，对于企业部署，平均每个开发者每天的活跃成本约为13美元，而每月支出则在150至250美元之间，且90%的用户每天成本保持在30美元以下。官网还建议，团队可以从小规模试点开始，并利用跟踪工具来建立基线，然后再进行更广泛的推广。

此前的页面存档显示，在4月16日之前，Anthropic的预估是平均每个开发者每天花费6美元，并且90%的用户每天成本低于12美元。对比之下，当前的预估值显著提升。

值得注意的是，代币（Tokens）是大型语言模型将词语分解为数字输入和输出的方式，它们是Anthropic Claude等AI聊天机器人运作的基本组成单位。

EZPR（一家科技和商业公关公司）的首席执行官Ed Zitron率先在X平台上指出了这一变化。
