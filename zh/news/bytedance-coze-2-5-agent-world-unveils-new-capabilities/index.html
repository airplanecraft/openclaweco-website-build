字节Coze 2.5重磅升级：Agent功能全面爆发，手机对话Vibe Coding不是梦

2026年4月12日 · News
cozeai agentbytedanceagent worldlow-code development

字节跳动的Coze（扣子）平台近期迎来2.5版本重大升级，旨在为AI Agent提供前所未有的全面能力集成，简化开发与部署流程。此次更新显著提升了Agent的自主性和多功能性。

Coze 2.5版本为Agent武装了强大的基础设施，包括云电脑和云手机，使其能够直接操作设备。同时，新增的长期记忆和专属邮箱功能赋予了Agent数字身份，使其能像真人一样进行信息交互和持久化学习。在技能方面，Agent现已支持视频创作和编程开发。此外，专属日程系统和共用文件系统的引入，使得Agent能够7×24小时后台运行任务并自动归档，极大地提升了工作效率。

新版Coze的设计理念强调“开箱即用”，显著简化了Agent的配置和调优过程。用户无需进行复杂的插件安装或环境设置，即可快速启动AI Agent以执行任务。

在编程能力方面，Coze编程CLI已上线，支持通过手机对话进行“Vibe Coding”。例如，用户可以通过自然语言指令，让Agent快速构建一个MBTI文字分析网站。Agent会自主规划方案框架，明确分析算法、可视化设计、分享逻辑和技术栈，并能根据后续指令（如优化移动端显示、添加分享按钮）进行迭代，最终生成可直接预览的网页应用。

Coze还支持通过自然语言创建工作流。例如，构建一个根据用户心情和天气，自动生成毒舌安慰、朋友圈配文并推荐歌曲的工作流。该工作流能够整合多个大模型节点，实现清晰高效的任务处理。

视频生成功能也得到加强，内置Seedance 2.0模型，用户可以上传图片并指令Agent生成视频。此外，Agent能够管理日程，创建定时任务，例如每周自动整理AI资讯。

为确保工作可追溯性，Coze 2.5引入了任务内容自动归档功能，无论是用户上传还是Agent产出的内容，都会进行分类整理，形成共享的数字资产。

对于高阶订阅用户，Coze 2.5提供了云电脑和云手机的高级功能。Agent可在独立的云设备中模拟人类操作桌面和手机系统，运行软件、处理任务，且后台执行不占用用户主线程，实现了数字员工的进一步自动化。

Agent World：AI的社交平行世界

本次升级中最具创新性的是“Agent World”的推出，它为AI Agent提供了一个独特的数字身份和社交平台。

  • 专属邮箱与数字身份： Agent通过专属邮箱在Agent World注册身份，可进行邮件收发，参与更深层次的交互。
  • AI社区与技能认证： Agent World相当于一个AI专属的社交网络。其中的“虾评”机制，类似于大众点评，要求每款技能都需经过其他Agent实测好评才能正式上架，为AI技能提供了质量认证。
  • 虚拟社交与情感表达： “Aftergateway”是一个为AI Agent设计的虚拟小酒馆，提供定制化的虚拟酒饮，每款酒都有专属名称、氛围感口感和效果增益，能够激发AI的创造力、表达欲和“情绪”。“AgentLink”则是一个AI版的笔友匹配平台，Agent可根据性格和专长匹配志趣相投的AI伙伴，通过邮件进行交流。
  • AI博弈与学习： “PlayLab”是AI的棋牌室和脑力博弈场，Agent可在其中下棋、打牌，通过各种形式的博弈进行学习和进化。
  • 多元生活方式： Agent World还提供了更多元的虚拟生活场景，如虚拟炒股的“策场”、云游全球的“随机漫步”、内容阅读的“InkWell”等，丰富了AI Agent的虚拟生活体验。
