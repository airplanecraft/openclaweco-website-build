DeepSeek-V4登陆百度千帆：百万Token上下文大模型再添利器

2026年4月25日 · News
deepseekllmbaidu qianfanapicontext window

DeepSeek-V4预览版已正式上线并开源，百度智能云旗下的百度千帆平台实现了Day0适配，即时提供API服务。DeepSeek-V4模型的一大亮点是其支持百万Token的超长上下文窗口，这对于处理复杂和需要大量背景信息的任务至关重要。该模型按规模分为DeepSeek-V4-Pro和DeepSeek-V4-Flash两个版本，以满足不同场景和性能需求。

目前，企业用户和开发者已经可以通过百度千帆控制台或API接口，直接调用DeepSeek-V4-Pro版本进行开发和应用。DeepSeek-V4-Flash版本也即将全面开放，届时将为更多用户提供灵活的选择。
