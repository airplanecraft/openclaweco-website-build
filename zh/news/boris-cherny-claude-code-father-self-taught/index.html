Claude Code之父Boris Cherny：经济学背景自学成才，铸就AI编程新时代

2026年4月12日 · News
claude codeai agentanthropicboris chernytypescript

作为Anthropic旗下明星产品Claude Code的灵魂人物，Boris Cherny的职业生涯充满传奇色彩。这位被誉为“Claude Code之父”的技术领袖，并非科班出身，而是从经济学专业背景，凭借自学能力，一步步杀入编程世界，并最终在AI时代掀起巨浪。

Cherny的“野路子”编程之路始于中学时代，他便开始为eBay商品列表编写网页代码，并为图形计算器开发程序。大学在加州大学圣地亚哥分校攻读经济学专业，后几经波折才完成学业。毕业后，他涉足互联网创业，随后加入了Meta（Facebook），在近七年时间里，他领导了Instagram服务器架构和开发基础设施，并致力于提升Meta的代码质量。期间，他撰写了TypeScript领域的权威著作《Programming TypeScript: Making Your JavaScript Applications Scale》，成为该语言的必读经典。

从Meta首席软件工程师跳槽至Anthropic后，Cherny不到一年便孵化出Claude Code。他透露，CC的诞生起初纯属偶然，作为新员工为了熟悉公司API，他开发了一个极简的命令行工具（CLI）。通过给模型添加批处理工具并将Python示例转换为TypeScript，Claude Code展现出惊人的能力：能准确执行`cat`命令读取文件，甚至能编写Apple Script查询电脑音乐播放器。Cherny将那一刻形容为“AGI之神降临”，模型仿佛本能地渴望使用工具。

Claude Code原型在Anthropic内部迅速普及，并经过多次迭代后推向用户。初期反响平平，直到202X年5月Anthropic首届开发者大会后台，Cherny将同期发布的Opus 4和Sonnet 4模型整合进Claude Code，使其代码生成比例从10%飙升至近30%。

在这一关键节点，Cherny曾与Claude Code产品负责人Catherine Wu短暂跳槽至Cursor。然而，由于Claude Code对Anthropic营收的巨大贡献（彼时营收暴涨5.5倍），两人在半个月后即被Anthropic重新请回。回归后的Cherny不负众望，推动Claude Code彻底商业化起飞，使其稳坐AI编程领域的头把交椅。得益于Claude Code的成功，Anthropic的年化收入已赶超OpenAI，而Claude Opus 4.5的发布则进一步巩固了Claude Code的市场地位。

除了Claude Code，Cherny还主导了今年初发布的Claude Cowork。这款产品与Claude Code一脉相承，甚至整个Cowork团队仅用10天就通过Claude Code生成了原型。不同于面向程序员的Claude Code，Cowork旨在赋能普通用户实现全自动工作，经授权后可直接操作用户电脑桌面、本地文件及各类办公软件。Cherny本人便是Cowork的忠实“狗粮”用户，他让Cowork处理日常事务，从审核同事周报、催促填写表格，到预订航班、支付罚单，甚至办理挖蛤蜊许可证，将自动化贯彻到极致。

然而，最近围绕Claude Code也出现了一些趣闻。其源代码被公开后，有评论指出其中存在不少“屎山代码”，引发了业界的讨论，令人不禁联想到这位自学成才的“野路子”是否对此有所“贡献”。

OpenMythos

OpenMythos 是 Anthropic Claude Mythos 模型的开源理论实现，由 kyegomez 基于公开研究和推测构建。它采用循环深度Transformer（RDT）架构，包含序曲（Prelude）、循环块（Recurrent Block）和尾声（Coda）三个阶段。其核心在于循环块，可进行多次迭代以实现深度推理，从而实现计算自适应和深度可变的推理能力。该模型支持MLA和GQA两种注意力机制，并使用稀疏MoE（Mixture of Experts）实现路由和共享专家。OpenMythos旨在通过更少的参数实现与更高参数量模型相当的性能。
