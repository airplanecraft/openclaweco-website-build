Nemp-memory
开源社区

Nemp Memory 是为 AI Agent（如 Claude Code 和 OpenClaw）设计的智能记忆插件。它通过简单的命令操作，实现项目环境的自动化学习与记忆管理，解决了传统 Agent 在会话间遗忘上下文的问题。其核心特点是100%本地运行，无需云服务、API密钥或复杂数据库，所有数据以纯 JSON 文件存储，保障隐私。Nemp Memory 提供智能上下文搜索、AI记忆建议、与 CLAUDE.md 双向自动同步等功能，极大简化了开发者的工作流程，确保 Agent 始终拥有最新且准确的项目上下文。

OpenMythos
开源社区

OpenMythos 是 Anthropic Claude Mythos 模型的开源理论实现，由 kyegomez 基于公开研究和推测构建。它采用循环深度Transformer（RDT）架构，包含序曲（Prelude）、循环块（Recurrent Block）和尾声（Coda）三个阶段。其核心在于循环块，可进行多次迭代以实现深度推理，从而实现计算自适应和深度可变的推理能力。该模型支持MLA和GQA两种注意力机制，并使用稀疏MoE（Mixture of Experts）实现路由和共享专家。OpenMythos旨在通过更少的参数实现与更高参数量模型相当的性能。
