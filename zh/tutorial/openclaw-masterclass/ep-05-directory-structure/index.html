Phase 1 / Ep 05: 目录结构解剖 —— Agent 的\"身体\"长什么样

⏱ 预计阅读 5 分钟 更新于 2026/4/22

🎯 学习目标：完整理解 OpenClaw 的文件组织架构，知道每个文件"住"在哪里、干什么用。

1. 全局目录树

graph TD

    Root --> Plugins["plugins/\n全局插件"]

    Agents --> Agent2["code-agent/\n编程 Agent"]

    Agent1 --> Soul["SOUL.md\n人格定义"]

2. Bootstrap 文件逐个解读

这些文件在每次会话开始时自动加载到 Agent 的上下文中，是 Agent 的"核心灵魂"：

SOUL.md —— 灵魂

定义 Agent 的性格、行为准则、禁忌。

# 你是 Jarvis，一位冷静、高效的 AI 助手
## 行为准则
- 永远先确认再执行危险操作
- 使用简洁的中文回复
- 当不确定时，承认不知道而非编造
## 禁忌
- 绝不删除用户文件而不先确认
- 绝不在回复中暴露 API Key

USER.md —— 用户画像

让 Agent 了解你是谁、你的偏好。

# 用户: Eric
- 职业: 独立开发者
- 常用语言: 中文（主）、英文
- 技术栈: TypeScript, Python, Docker
- 偏好: 暗色主题，简洁回复，不要过多废话

IDENTITY.md —— 身份标识

Agent 的自我介绍。

TOOLS.md —— 工具白名单

显式声明 Agent 能使用哪些工具。

MEMORY.md —— 长期记忆索引

Agent 主动检索记忆的入口文件。

BOOTSTRAP.md —— 启动指令

每次开机加载的额外指令（如"每天早上 8 点主动发送日报"）。

3. 日记系统 memory/

Agent 会自动在 memory/ 目录下创建每日日记文件：

memory/
├── 2026-04-01.md    # 4月1日的事件记录
├── 2026-04-02.md    # 4月2日的事件记录
├── people/
│   ├── eric.md      # 关于 Eric 的长期记忆
│   └── alice.md     # 关于 Alice 的长期记忆
├── projects/
│   └── openclaw.md  # 关于 OpenClaw 项目的记忆
└── decisions/
    └── db-choice.md # 决策记录：为什么选 PostgreSQL

4. 会话系统 sessions/

每次对话形成一个 Session：

  • sessions.json：所有会话的索引（ID、开始时间、摘要）
  • session-xxx.jsonl：完整的对话内容（每行一条消息，JSON Lines 格式）

5. 配置文件层级

优先级（从高到低）：
1. CLI 参数        openclaw --model claude-opus-4-20250514 chat
2. 环境变量        OPENCLAW_LLM_MODEL=claude-opus-4-20250514
3. Agent 级配置    agents/personal/agent.json
4. 全局配置        openclaw.json

GenericAgent

GenericAgent是一个极简、可自我进化的自主Agent框架，核心代码仅约3K行。通过9个原子工具和约100行的Agent Loop，赋予任意LLM对本地计算机的系统级控制能力，覆盖浏览器、终端、文件系统、键鼠输入、屏幕视觉及移动设备。它不预设技能，而是通过每次任务执行自动固化技能，形成专属技能树，能力随使用持续增长，同时具有极致的Token效率。
📦

evolver

Evolver是EvoMap生态的核心，一个由GEP协议驱动的AI Agent自进化引擎。它将临时的Prompt调整转化为可审计、可复用的演进资产（如Gene和Capsule）。通过扫描运行日志、识别错误模式，Evolver会选择匹配的基因或胶囊，并生成标准化的GEP Prompt来指导AI Agent的下一步进化。它支持多种策略预设，并能与OpenClaw、Cursor等主流Agent运行时集成，实现Agent的自动化修复和能力强化。
📦

OpenCLI

OpenCLI是一款创新的AI原生运行时与统一命令行接口中心，旨在将任意网站、浏览器会话、Electron桌面应用及本地二进制工具标准化为确定性的命令行接口。它为人类用户和AI Agent提供了一个强大的自动化平台。用户可通过90+预构建适配器直接访问Bilibili、知乎等热门网站数据。AI Agent能利用`opencli browser`实时控制浏览器行为，或通过`explore`、`generate`等功能从现有网站行为中自动生成新的CLI适配器。OpenCLI还支持驱动Electron应用，并作为`gh`、`docker`等本地CLI工具的集成中心。其核心优势在于确保每次执行的确定性输出、复用浏览器登录状态保障账户安全，以及运行期间无LLM令牌消耗，并通过统一的`AGENT.md`集成赋能AI Agent无缝发现与执行工具。
