🪨 Caveman 完全指南：让 AI Agent 省 75% Token 的终极武器

从安装到精通，完整掌握 GitHub 上 42k+ Star 的 Caveman 生态系统。覆盖 Claude Code、Antigravity、Gemini CLI 三大平台的安装配置，深入解析 Hooks 自动激活机制，逐一演练 5 大 Skill（caveman / caveman-commit / caveman-review / caveman-compress / caveman-help），配合 Mermaid 流程图和实战代码示例。让你的 AI Agent 说话像原始人——但脑子比谁都清醒。

开始学习

课程目录

01

第 01 期 | Caveman 是什么 — Token 压缩的哲学与生态全景

理解 Caveman 的核心理念：用更少的 Token 说更精准的话。掌握 Caveman 生态系统三件套（caveman / cavemem / cavekit）的定位与协作方式。
02

第 02 期 | 五大平台安装实战 — Claude Code / Antigravity / Gemini CLI / Codex / OpenCode

分平台完整演示 Caveman 的安装与验证流程。对比五大主流 AI Agent 平台（Claude Code、Antigravity、Gemini CLI、Codex、OpenCode）的安装命令、配置文件、自动激活机制，附带 Cursor / Windsurf 等扩展安装速查。
03

第 03 期 | Hooks 深度解析 — Caveman 的自动激活引擎

深入剖析 Caveman 的三个 Hook 文件（caveman-activate.js / caveman-mode-tracker.js / caveman-statusline.sh），理解 SessionStart 和 UserPromptSubmit 的生命周期，掌握 Flag File 桥接机制。
04

第 04 期 | 四档变速 — 从 Lite 到 Ultra 到文言文

逐一体验 Caveman 的 7 个强度级别（lite / full / ultra / wenyan-lite / wenyan / wenyan-ultra），通过同一个技术问题的不同回答风格，找到最适合你的压缩档位。
05

第 05 期 | /caveman 核心 Skill — 日常开发的压缩利器

深入解析 Caveman 主技能的 Skill 文件结构、触发方式和回答模式。通过 Node.js 调试、TypeScript 泛型解释、代码重构三大实战场景，展示 Caveman 在日常开发中的真实表现。
06

第 06 期 | /caveman-commit — 让 Git 提交信息回归本质

掌握 caveman-commit 技能的使用方法和规则：Conventional Commits 格式、≤50 字符主题行、'Why over What' 原则。对比五大平台的 commit 工作流，并展示如何与 Git Hook 自动集成。
07

第 07 期 | /caveman-review — 一行式代码审查

掌握 caveman-review 技能的审查格式（行号 + 严重级别 + 一句诊断），学会用它高效审查 PR 和代码改动。对比五大平台的代码审查工作流，并展示如何与 GitHub Actions 集成实现自动化审查。
08

第 08 期 | /caveman:compress — 压缩你的 CLAUDE.md，每次省 46% 输入 Token

掌握 caveman-compress 技能的使用方法和安全规则。理解输入 Token 压缩的原理：为什么每次会话启动都在浪费 Token 读取冗长的上下文文件，以及 compress 如何在不丢失信息的前提下解决这个问题。
09

第 09 期 | /caveman-help 与 Always-On 配置 — 让 Caveman 无处不在

掌握 caveman-help 快速参考卡的使用，并在五大平台上配置 Always-On 永久激活。深入理解各平台的规则文件系统差异，学会将 Caveman 配置提交到 Git 仓库实现团队共享。
10

第 10 期 | 性能基准与最佳实践 — 把 Caveman 融入你的开发工作流

用真实数据证明 Caveman 的压缩效果，运行官方 Benchmark 和 Eval 套件。构建完整的 Caveman 日常开发工作流：从会话启动到代码审查到 Git 提交的全链路最佳实践。
